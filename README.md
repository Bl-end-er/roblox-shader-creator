# Roblox Shader Creator

A demo created inside of Roblox, allowing users to create displays that are programmable to display shaders.

## Overview

This project demonstrates how to create interactive, programmable shader displays within Roblox. Users can experiment with different visual effects and see the results in real-time.

## Getting Started

You can explore this project in two ways:

1. **Open the provided file** – The main project file contains all the necessary assets and scripts to run the demo directly in Roblox Studio.
2. **Explore the scripts** – Check the subfolders to look through each individual script and see how the shader system is implemented.

## Features

- Programmable shader displays
- Designed for experimentation and learning

## Usage

1. Open the project in Roblox Studio.
2. Run the demo to see the shader displays in action.
3. Use the provided GUI to programme your selected shaders.
4. Modify or create your own shaders by exploring the scripts in the subfolders.
